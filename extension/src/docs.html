Mermaid Markdown Memo の使い方（やさしい版） - 2025/12/29 時点

このアプリでできること

これは「Markdown（マークダウン）」でメモを書き、必要に応じて Mermaid（図を描くための記法）をプレビューできるメモアプリです。

  • メモを作る / 消す / 切り替える
  • Markdownで見出しや箇条書きが書ける
  • Mermaidでフローチャートなどの図が作れる
  • タブで「編集」と「プレビュー」を切り替えられる

基本の使い方

1. 新しいメモを作る

左上の「＋」ボタンで新しいメモが作れます。

2. メモを切り替える

左の一覧からメモをクリックすると、そのメモが開きます。

3. 編集 / プレビュー

タブで開いた画面では「編集」「プレビュー」を切り替えられます。 編集は文字を書く画面、プレビューは見た目を確認する画面です。

Markdownの例

例えば、箇条書きは次のように書けます。

- りんご
- みかん
- ぶどう

さらに、箇条書きの行末でEnterを押すと、次の行に自動で「- 」が入るようにしています。

Mermaidの例（図を描く）

次のように書くと、プレビューで図になります。

よくある質問

Q. データはどこに保存されますか？

この拡張の中（Chromeのローカル保存領域）に保存されます。

Q. ネットに送信されますか？

このアプリは外部サーバーに送信する仕組みを持ちません。

安全性とデータ保存（大事）

ここは「安心して使えるか」を判断するための説明です。 この説明は 2025/12/29 時点で、拡張の実装（manifest.json / storage.js）と公式ドキュメントを元にしています。

この拡張が持っている権限

  • 権限は storage のみ です（メモを保存するため）。
  • サイトの読み取りや変更（host_permissions）は要求しません。

データはどこに保存される？

メモは Chrome 拡張の保存領域である chrome.storage.local に保存されます。 公式ドキュメントでは、local は「PC内に保存され、拡張を削除すると消える」と説明されています。

保存しているデータの中身（保存キー）

この拡張は、次のキー名で保存しています（storage.js を根拠）。

  • notes：メモ本体（タイトル/本文/更新日時など）
  • noteSortOrder：左の一覧の並び順
  • sortDirection：並び替えの昇順/降順

ネット送信はある？

  • この拡張は 外部サーバーへ送る処理を持っていません（通信の仕組みを前提にしていません）。
  • 保存先はあくまで あなたのChrome内 です。

注意点（より正直な説明）

  • 拡張の保存領域は暗号化された保管庫ではありません。 公式のプライバシーガイドでも、機密データを拡張ストレージに保持しない注意喚起があります（該当ページは最終更新が古いので、考え方として参照）。
  • この拡張は「パスワード」や「秘密鍵」などの保管用途には向きません。

自分のデータを確認/削除する方法

  • 確認：DevTools の Application > Storage > Extension Storage から確認できます。 （公式: Chrome DevTools の Extension Storage 画面）
  • 削除：Chrome の「拡張機能を管理」画面からこの拡張を削除すると、localのデータも消えます。 （公式: Chrome ウェブストア ヘルプ「拡張機能を管理」）
